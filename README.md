# MERN Stack To-Do App

A simple full-stack To-Do List application built using the MERN stack (MongoDB, Express.js, React, Node.js). This app allows users to add, view, and delete tasks.

---


# Backend SetUp
--- cd backend
---npm install 
--- npm  run dev

# FrontEnd setUp
--cd my-react-app
---npm install
--npm run dev 

##  Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **HTTP Client:** Axios
